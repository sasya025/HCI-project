# 🎉 EVENT MANAGERS

## 📌 Project Description
This project is built using **HTML, CSS, and JavaScript**.  
It allows users to select different events and choose services based on their needs.

The total cost of selected services is dynamically displayed on the side.

## 🚀 Features
- Select different types of events
- Choose multiple services
- Real-time cost calculation
- Simple and user-friendly interface

## 🎯 Use Cases
From small birthday parties to grand weddings,  
this system helps in organizing and planning events efficiently.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
